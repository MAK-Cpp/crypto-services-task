# Java Service

Сервис реализует:

- электронная подпись стандарта PKCS#7 переданного документа (присоединенная или отсоединенная)
- проверка электронной подписи стандарта PKCS#7 переданного документа (присоединенная или отсоединенная)
- устанавливает защищенное доверенное соединение с каким-либо конкретным внешним ресурсом по HTTPS, загружает документ,
  отдает клиенту.

Используемый стек:

- Язык: Java 21
- Фреймворк: Spring Boot 4.1.0
- Контейнер сервлетов: embedded Apache Tomcat через Spring MVC
- Криптографические провайдеры:
    1. Bouncy Castle - для электронной подписи стандарта PKCS#7 переданного документа

## Как запустить сервис

### IJ IDEA Ultimate

1) Открыть `Configuration -> Edit... -> Spring Boot`
2) `Modify options -> Working directory`, поставить путь до корня проекта
3) `Modify options -> Environment variables`, указать путь к .env файлу или вручную заполнить нужные переменные среды
