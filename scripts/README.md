# Scripts

Здесь хранятся удобные скрипты, которые облегчают работу и тестирование с проектом.

## [generate_pkcs12.py](generate_pkcs12.py)

Скрипт позволяет сгенерировать keyset для приложения.

Пока что скрипт поддерживает только тип PKCS12, в будущем при необходимости можно расширить.

### Алгоритм

1) Генерирует приватный ключ
2) Генерирует сертификат X.509
3) Сериализует ключ и сертификат, запаролив их паролем
4) Сохраняет результат в файл

### Как запускать

1) Создать .env файл в корне репозитория, заполнить переменные
    ```
    KEYSTORE_PATH
    KEYSTORE_TYPE
    KEYSTORE_PASSWORD
    KEYSTORE_ALIAS
    ```
2) Запустить скрипт, рабочая директория - корень репозитория.
